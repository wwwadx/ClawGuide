- `github` - GitHub 集成

## 如何添加新场景

当遇到新的场景时：

1. 记录用户的原话
2. 分析核心需求
3. 设计解决方案
4. 列出需要的 skills
5. 估算配置时间和效果
6. 添加到这个文件

模板：

```markdown
### [场景名称]
**用户说：** "[用户的具体描述]"

**推荐：** [方案名称]
- [好处 1]
- [好处 2]
- [好处 3]

**需要的 Skills：**
- `skill-1`
- `skill-2`

**配置时间：** X 分钟
**效果：** [具体效果描述]
```
