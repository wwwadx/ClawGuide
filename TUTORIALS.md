# OpenClaw 新手引导系统

> 收集和整理 OpenClaw 的实际用例，帮助新用户快速上手

## 我们的使命

解答 "What can I do with OpenClaw?" 这个问题，通过：
1. 收集真实世界的用例和教程
2. 理解用户的工作场景
3. 匹配合适的 skills 和工作流
4. 引导用户实现自动化

## 工作流程

```
用户描述场景 → 理解需求 → 匹配用例 → 推荐方案 → 安装配置 → 验证效果
```

## 已收集的真实用例

### 1. PR 审查自动化 (@bangnokia)
**场景：** 代码审查
**工具：** GitHub + Telegram + coding-agent
**效果：** 打开 PR → 自动审查 → Telegram 反馈建议和合并决策

### 2. 酒窖管理 (@prades_maxime)
**场景：** 个人收藏管理
**工具：** CSV + 自定义 skill
**效果：** 962 瓶酒的本地查询和管理系统

### 3. 超市自动购物 (@marchattonhere)
**场景：** 生活自动化
**工具：** Browser automation
**效果：** 每周 meal plan → 添加商品 → 预订配送 → 确认订单

## 如何使用这个系统

### 对于新用户：
1. 告诉我你的工作场景（例如："我每天要处理很多邮件"）
2. 我会推荐相关用例和 skills
3. 我会帮你安装和配置
4. 我们一起验证效果

### 对于贡献者：
- 添加新的用例到这个文件
- 记录你的 setup 过程
- 分享你的 workflow
